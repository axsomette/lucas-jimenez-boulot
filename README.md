# MonPortfolio

Portfolio personnel de développeur — construit avec Astro 5 + TypeScript, déployé sur GitHub Pages.

## Démarrage rapide

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
# → http://localhost:4321

# Build production
npm run build

# Preview du build
npm run preview
```
