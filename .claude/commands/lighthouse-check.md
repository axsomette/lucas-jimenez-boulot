# Vérification Lighthouse

Lance un audit de performance et accessibilité avant déploiement.

## Usage
/lighthouse-check

## Instructions
1. `npm run build` pour générer le build de production
2. `npm run preview` pour lancer le serveur de preview
3. Vérifier mentalement : images optimisées, lazy loading, meta tags, aria labels
4. Lister les points d'amélioration potentiels pour atteindre > 95 sur toutes les métriques
