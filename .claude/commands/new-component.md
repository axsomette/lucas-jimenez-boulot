# Créer un nouveau composant Astro

Crée un composant Astro typé dans `src/components/`.

## Usage
/new-component [nom] [type: ui|section]

## Instructions
1. Créer le fichier `src/components/[type]/[Nom].astro`
2. Ajouter le frontmatter TypeScript avec les props typées (interface Props)
3. Structure HTML sémantique avec aria-labels
4. CSS scoped dans `<style>` avec custom properties
5. Exporter le composant et l'ajouter dans l'index si applicable
